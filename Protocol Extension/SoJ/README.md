## 1.先用class實作個酒精飲料
### 用class實作出來不算太難，但是在管控上就顯得很困難！！
有些實作內容其實是重複的，應該寫成一包管控，這樣之後在維護上也會比較輕鬆，未來要新增新的飲品也才不會 "少調用" 或 "調用錯方法"
